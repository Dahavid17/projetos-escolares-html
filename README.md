# Projetos Escolares em HTML e CSS

Repositório contendo atividades e projetos desenvolvidos em sala de aula
para aprendizado de desenvolvimento web.

## Tecnologias utilizadas
- HTML
- CSS

## Sobre os projetos
Os projetos têm como objetivo praticar a criação de páginas web,
estruturação de conteúdo e estilização básica utilizando CSS.

## Observação
Projetos desenvolvidos para fins educacionais e de estudo.
